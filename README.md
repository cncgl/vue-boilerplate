vue-boilerplate
======================

Vue.js boilerplate based on [re-fort/vue-webpack-boilerplate](https://github.com/re-fort/vue-webpack-boilerplate)

## What's included
- Vue(1.x.x)
- webpack
- vue-loader
- vue-resource
- vue-router
- Bulma
- Font Awesome

## Build
```
$ npm run build
```

## Usage
```
$ npm run dev
```
visit http://localhost:3000/
